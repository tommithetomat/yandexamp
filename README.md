# YandexAmp ⚡

Десктопный плеер в стиле классического **Winamp 2.x** со стримингом из **Яндекс Музыки**.

![Electron](https://img.shields.io/badge/Electron-43-2b2e3a) ![Platform](https://img.shields.io/badge/Windows%20%7C%20macOS-0078d4) ![License](https://img.shields.io/badge/license-MIT-green)

## Возможности

- 🎨 Классический интерфейс Winamp: LCD-дисплей, спектроанализатор, эквалайзер, плейлист
- 🎛 10-полосный эквалайзер с пресетами (Rock, Pop, Classic, Bass)
- 🌈 5 скинов: Classic Green, Blue Steel, Amber, Yandex, Blood — кнопка в титлбаре
- ✨ 3 визуализатора: спектр, осциллограф, звёздное поле — клик по дисплею
- 🔐 Вход через браузер (OAuth), по логину/паролю или токеном
- 📚 Свои плейлисты + подборки Яндекса: Плейлист дня, Премьера, Дежавю
- ❤️ «Мне нравится» — все лайкнутые треки
- ⚡ «Моя волна» — бесконечный поток: новые треки подгружаются автоматически
- 🔀 Shuffle, Repeat, поиск по всей Яндекс Музыке

> Для прослушивания нужна активная подписка Яндекс Плюс.

## Установка (для пользователей)

### Windows

1. Скачайте `YandexAmp-Setup-*.exe` из [Releases](../../releases)
2. Запустите — установщик сам создаст ярлык на рабочем столе
3. SmartScreen может предупредить о неизвестном издателе — «Подробнее → Выполнить в любом случае»

### macOS

1. Скачайте `YandexAmp-*-arm64.dmg` (Apple Silicon) или `YandexAmp-*-x64.dmg` (Intel) из [Releases](../../releases)
2. Откройте DMG и перетащите YandexAmp в Applications
3. Приложение не подписано сертификатом Apple, поэтому при первом запуске: **правый клик → Открыть**. Если macOS пишет «приложение повреждено», выполните в терминале:
   ```bash
   xattr -cr /Applications/YandexAmp.app
   ```

## Запуск из исходников (для разработчиков)

Работает на Windows и macOS:

```bash
git clone <repo-url>
cd yandexamp
npm install
npm start
```

## Сборка установщиков

```bash
npm run dist       # Windows: dist/YandexAmp-Setup-<версия>.exe
npm run dist:mac   # macOS (только на маке): dist/YandexAmp-<версия>-<arch>.dmg
```

DMG собирается только на macOS — но для этого есть CI: при пуше тега `v*` GitHub Actions сам собирает установщики под обе платформы и прикладывает их к релизу:

```bash
git tag v1.0.0
git push origin v1.0.0
```

## Как это устроено

- **Electron** — безрамочное окно, стилизованное под Winamp
- **Web Audio API** — цепочка BiquadFilter'ов для эквалайзера + AnalyserNode для визуализаторов
- **Неофициальное API Яндекс Музыки** (`api.music.yandex.net`) — поиск, плейлисты, ротор «Моей волны», прямые ссылки на потоки
- Токен авторизации хранится только в памяти процесса, никуда не передаётся

## Дисклеймер

Проект не аффилирован с Яндексом. Использует неофициальное API — работоспособность может сломаться при изменениях на стороне сервиса. Только для личного использования.
